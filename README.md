Skill Cloud
=============================
This is visializing skill of namikingsoft like cloud.  
http://blog.namiking.net/skillcloud/


Get Started
-----------------------------

```sh
npm install
npm start
open http://localhost:1234
```

Development
-----------------------------

#### Build
```sh
npm run build
```

#### Deploy
```sh
npm run deploy
```

#### Test Watch
```sh
npm test
```

#### Test Single
```sh
npm run test:single
```
